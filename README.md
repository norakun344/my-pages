# my-pages

個人プロフィールページです。

## 機能

- ✨ Appleフロストグラス風のLiquidGlassデザイン
- 🎨 ダークモード対応
- 📊 アクセスカウンター
- 🔄 マウス追従3D効果（プロフィール画像）
- 📱 レスポンシブデザイン
- 🔗 SNS共有機能（Twitter/LINE/コピー）
- 🌐 OGP実装済み

## OGP対応

ページがSNSでシェアされた時に、以下の情報が表示されます：

- **タイトル**: @norakun344
- **説明**: My skills: HTML, CSS | Games: Fortnite, Minecraft, GTA5, Among us
- **画像**: プロフィール画像（IMG_2736.JPG）
- **URL**: https://mypage.norakun344-github.f5.si/

Twitter Card対応で、全てのSNSプラットフォームで最適なプレビューが表示されます。

## 技術スタック

- HTML5
- CSS3（Glassmorphism、3D transforms）
- JavaScript（モウストラッキング、ローカルストレージ）

## ファイル構成

- `index.html` - メインページ
- `style.css` - スタイル定義
- `script.js` - インタラクティブ機能
- `CNAME` - カスタムドメイン設定
